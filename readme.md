## caddy-compose

1.  clone
    ```
    git clone https://github.com/takamafa/caddy-compose.git
    ```

2.  up
    ```
    docker-compose up -d
    ```

3.  test
    - when execute
      ```
      curl -k https://localhost/.gitignore
      ```
    - then see
      ```
      *
      ```

4.  down
    ```
    docker-compose down
    ```
